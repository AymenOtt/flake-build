OCI-CTL-LIST(8)
===============

NAME
----

**oci-ctl list** - List registered flake applications

SYNOPSIS
--------

.. code:: bash

   USAGE:
       oci-ctl list

   OPTIONS:
       -h, --help       Print help information
       -V, --version    Print version information


DESCRIPTION
-----------

List registered flake applications.

FILES
-----

* /usr/share/flakes

EXAMPLE
-------

.. code:: bash

   $ oci-ctl list
